Basic System Commands

pwd                           → Show current working directory
whoami                        → Show current logged-in user
hostname                      → Show system hostname
date                          → Display system date/time
date +%format                 → Custom date format
clear                         → Clear terminal
cal                           → Show calendar
uptime                        → Show system uptime with load average
uname -a                      → Show kernel and system info
arch                          → Show machine architecture
lscpu                         → CPU info
lsblk                         → List block devices & partitions
cat /etc/os-release           → Show OS version



Files & Directory Management

ls                            → List directory contents
ls -lt                        → Sort by time (newest first)
ls -ltr                       → Sort by time (oldest first)
ls -lh                        → Long format with readable size
cd /path                      → Navigate to directory
cd ..                         → Go to parent directory
mkdir <dir>                   → Create directory
rmdir <dir>                   → Remove empty directory
rm -rf <dir>                  → Delete directory and all contents
touch <file>                  → Create or update timestamp
cp <source> <dest>            → Copy file
cp file1 newname              → Duplicate file
mv <src> <dest>               → Move file
mv old new                    → Rename file
find /path -name file         → Search for file
locate <file>                 → Fast search (after updatedb)



View & Edit Files

cat <file>                    → View file content
less <file>                   → View with scroll/search
more <file>                   → View page by page
nano <file>                   → Edit using nano editor
vi <file>                     → Open file in vi editor
head -n <file>                → First N lines
tail -n <file>                → Last N lines
wc -l <file>                  → Count lines
cmp file1 file2               → Compare files
diff -u file1 file2           → Show differences



Sorting & Filtering Text

sort <file>                   → Sort lines
sort -r <file>                → Reverse sort
sort <file> | uniq           → Unique lines only
grep "text" <file>            → Search within file
egrep "word1|word2" <file>    → OR search
shuf <file>                   → Randomize lines



Text Transformation Utilities

awk -F, '{print $2}' file.csv → Extract 2nd column
cut -c1-2 file.txt            → Extract characters by position
sed -n "5p" file.txt          → Print 5th line only
sed -n 's/from/to/g' file.txt → Replace text (print output)
tr [:lower:] [:upper:]        → Convert lowercase → uppercase
tr [:punct:] Z                → Replace punctuation with Z
tr -d Z                       → Delete character Z
truncate -s 100M file.txt     → Resize file to 100MB
echo "ABC" | fold -w1         → Print each character on new line



Archive & Compression Commands

gzip -k file                  → Compress and keep original
gunzip file.gz               → Decompress
tar -czf archive.tar.gz dir  → Create compressed tar archive
tar -xzf archive.tar.gz      → Extract tar archive
zip archive.zip file1 file2  → Create zip archive
unzip archive.zip            → Extract zip



Downloading Files

wget URL                      → Download using wget
wget -O filename URL          → Download and rename
curl URL                      → Fetch URL data / API call



Package Management

apt install pkg              → Install package (Debian/Ubuntu)
yum install pkg              → Install package (CentOS)
dnf install pkg              → Install package (Fedora/RHEL8+)
rpm -qa | grep <pkg>         → Query installed packages
dnf list installed           → List installed packages
apt search <pkg>             → Search in repo
yum/dnf list available       → Show available packages



Permissions & Ownership

ls -ltr                      → View permissions sorted by time
chmod a+rwx file             → Give full permission to all
chmod 755 file               → rwxr-xr-x
chown user file              → Change owner
chgrp group file             → Change group
umask                        → Default permissions for new files



User & Group Management

useradd username             → Create user
passwd username              → Set password
userdel username             → Delete user
groupadd grp                 → Create group
groupdel grp                 → Delete group
id username                  → Show UID/GID info
su user                      → Switch user
exit                         → Exit user session
sudo command                 → Execute as admin



Process & System Monitoring

ps -ef | grep process        → List running processes
pgrep process                → Get process ID
kill -9 PID                  → Force kill
pkill process                → Kill by name
top                          → Real-time system monitor
free -h                      → RAM usage
df -h                        → Disk usage
du -sh *                     → Size of folders/files
du -ah /var | sort -rh | head -5 → Biggest files in /var



Services (systemd)

systemctl status service     			→ Check service status
systemctl start service      			→ Start service
systemctl stop service       			→ Stop service
systemctl restart service    			→ Restart service
systemctl list-units --type=service --all 	→ List all services
journalctl -xe               			→ View logs
systemctl enable   				→ enable service on boot


Jobs & Background Tasks

nohup script.sh &            → Run script in background
jobs                         → List background jobs
bg                           → Resume job in background
fg                           → Bring job foreground
at time cmd                  → Schedule one-time task
crontab -e         	     → Edit cron jobs
crontab -l                   → List cron jobs



Networking

ifconfig                     → Network info
ping site.com                → Check connectivity
curl ifconfig.me             → Show public IP
traceroute domain            → Trace route to target
nslookup domain              → DNS lookup
dig domain                   → Detailed DNS info
telnet IP port               → Test port
ssh user@host                → Remote login
scp file user@host:/path     → Transfer file via SSH
netstat -tulnp               → Show listening ports
ss -tulnp                    → Modern socket listing
